# ExkutorLang v1.0

A strictly typed, secure, minimalist domain-specific language for executing shell commands on Windows 11 via WSL2 and PowerShell.

## Quick Start

1.  **Clone**:
    ```bash
    git clone https://github.com/org/ExkutorLang.git
    cd ExkutorLang
    ```
2.  **Build**:
    ```bash
    python build.py
    ```
3.  **Run**:
    ```bash
    python exkutor.py path/to/script.exkutor
    ```
